## What's this

Set module when new file create.
See examples below.

```
$ vim lib/hoge/fuga.rb
# auto inserted
#
# module Hoge::Fuga
# end
```

## Install
```
Plug 'nishisuke/ruby-module-new-vim'
" or
NeoBundle 'nishisuke/ruby-module-new-vim'
" or some plugin maneger
```
